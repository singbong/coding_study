백준 9095: 1,2,3 더하기

학교: 대진대학교   전공: AI 빅데이터   학번: 20191624   이름: 신봉균  


문제




풀이



이 문제는 DP(Dynamic Programming) 문제이다. 이 문제를 풀 때 DP 접근을 위해 패턴이나 규칙을 찾기 위해 경우의 수를 풀어서 썼다.

1 -> (1) -> 1개
2 -> (1+1),(2) -> 2개
3 -> (1+1+1),(1+2).(2+1),(3) -> 4개
4 -> (1+1+1+1),(1+1+2),(1+2+1),(1+3),(2+1+1),(2+2),(3+1) ->7개 5 -> (1+1+1+1+1),(1+1+1+2),(1+1+2+1),(1+1+3),(1+2+1+1),(2+1+1+1),(1+2+2),(2+1+2),(2+2+1),(1+3+1),(3+1+1),(2+3),(3+2) ->13개

위의 규칙을 봤을때, 4번째 경우의 수는 첫 번째와 두 번째, 세 번째 경우의 수를 합한 것과 같다는것, 5번째의 경우의 수는 2,3,4 번재 경우의 수의 합과 같다는 것을 볼 수 있다.

따라서, 위의 규칙을 보면 하나의 점화식이 나온다.


N이 3보다 클때(n>3), f(n)= f(n-1) + f(n-2) + f(n-3)
